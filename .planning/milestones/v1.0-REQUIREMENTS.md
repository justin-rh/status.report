## v1 Requirements — Final Status

### Data Collection

- [x] **COLL-01**: Tool parses the PC hostname and decodes city, device type (warehouse workstation, user-assigned laptop, department laptop, P3), department code, company code, and station number using the Master Electronics naming convention — *Validated Phase 1: 26-test suite, all 21 city codes covered*
- [x] **COLL-02**: Tool collects hardware stats: CPU model, total RAM, disk capacity and free space, Windows OS version and build number — *Validated Phase 2: WMI + psutil collectors with graceful degradation*
- [x] **COLL-03**: Tool enumerates all local user profiles on the machine (not just the currently logged-in user) — *Validated Phase 2: HKLM ProfileList winreg enumeration*

### App Detection

- [x] **APP-01**: Tool detects whether NinjaRMM / NinjaOne agent is installed (registry-based, all four Uninstall key paths) — *Delivered Phase 4: keyword sweep across HKLM/HKLM\Wow6432Node/HKCU/HKCU\Wow6432Node. Live machine test deferred (acknowledged at close)*
- [x] **APP-02**: Tool detects whether CrowdStrike Falcon is installed and its service is present — *Delivered Phase 4: DisplayName 'CrowdStrike Windows Sensor'/'CrowdStrike Sensor Platform' confirmed from live registry. Service state via SCM query. Live machine validation test deferred (acknowledged at close)*
- [x] **APP-03**: Tool detects whether MERP (Master Electronics ERP) is installed — *Delivered Phase 4: filesystem-first at PVX Plus hardcoded path; registry search for version on hit*
- [x] **APP-04**: Tool detects whether Microsoft 365 apps are installed — *Delivered Phase 4: single 'Microsoft 365' suite entry per D-05 decision (vs 5 individual entries per original spec). Stakeholder sign-off deferred (acknowledged at close)*
- [x] **APP-05**: Tool detects whether Zoom is installed — *Delivered Phase 4: keyword sweep across all 4 Uninstall paths*
- [x] **APP-06**: Tool detects whether Google Chrome is installed — *Delivered Phase 4: keyword sweep across all 4 Uninstall paths*
- [x] **APP-07**: Tool detects whether Claude desktop app is installed — *Delivered Phase 4: MSIX AppModel repository family prefix (primary) + Uninstall sweep (fallback)*

### Output

- [x] **OUT-01**: Tool generates an HTML character sheet with RPG/D&D-influenced aesthetic — stat block layout for hardware, class/guild/realm fields for device identity, equipment list for installed apps — *Delivered Phase 3: dark navy character sheet with HP bar, color-coded badges, quest status banner. Visual browser verification deferred (acknowledged at close)*
- [x] **OUT-02**: HTML file is saved to the same directory the .exe was launched from (flash drive) — *Validated Phase 3+5: derived from `Path(sys.executable).parent`*
- [x] **OUT-03**: Tool handles unrecognized or non-conforming hostnames gracefully — *Validated Phase 1: returns device_type=Unknown with raw hostname, no exception*

### Packaging

- [x] **PKG-01**: Tool is packaged as a Windows .exe using PyInstaller `--onedir` mode, runnable without installation or admin rights on Windows 10/11 — *Validated Phase 5: CrowdStrike Falcon test passed 2026-05-05*
- [x] **PKG-02**: All output written to flash drive only — no writes to host PC filesystem, registry, or %TEMP% — *Validated Phase 5: confirmed during USB validation*
