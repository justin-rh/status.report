   ```python
       """Populate uptime_seconds via psutil.boot_time(). Never raises (D-05)."""
       try:
           report.uptime_seconds = int(time.time() - psutil.boot_time())
       except Exception as exc:
           report.collection_errors.append(f"Uptime collection failed: {exc}")
   ```

5. Add `collect_pending_updates()` as a new public function after `collect_profiles()` and before the private helpers section. Use this exact code:
   ```python
   def collect_pending_updates(report: AuditReport) -> None:
       """Populate pending_updates via WUA COM. Windows-only. Never raises.

       Degrades to pending_updates=None when _WIN32COM_AVAILABLE is False (CI)
       or when the caller lacks SYSTEM/Administrator privilege (D-09, D-10).
       """
       if not _WIN32COM_AVAILABLE:
           return
       try:
           session = _win32com_client.Dispatch("Microsoft.Update.Session")
           searcher = session.CreateUpdateSearcher()
           result = searcher.Search("IsInstalled=0 and Type='Software'")
           report.pending_updates = result.Updates.Count
       except Exception as exc:
           report.collection_errors.append(f"Pending updates collection failed: {exc}")
   ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| psutil → OS kernel | psutil.boot_time() reads kernel boot time; standard user privilege, no elevation required |
| win32com → WUA COM server | COM dispatch to Microsoft.Update.Session; requires SYSTEM/Administrator; standard user call raises COMError |
| `scry.exe` → flash drive (output) | All artifacts written to USB only; no host PC writes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-01 | Denial of Service | `collect_pending_updates` — WUA COM call | mitigate | Outer `except Exception` catches all COM errors including pywintypes.com_error; pending_updates stays None; tool continues |
| T-13-02 | Denial of Service | `_collect_uptime` — psutil.boot_time() | mitigate | Wrapped in try/except; uptime_seconds stays None on failure; collection_errors logged |
| T-13-03 | Elevation of Privilege | win32com.client import | mitigate | `_WIN32COM_AVAILABLE` guard; import inside try/except; never imported unconditionally; WUA COM call never made when guard is False |
| T-13-04 | Tampering | `pywin32==311` in requirements.txt | accept | Pinned exact version; low-value local tool; no network calls made by this tool |
| T-13-05 | Information Disclosure | WUA COM session exposes update titles | accept | Only `result.Updates.Count` (integer) is read; no update titles or KB numbers surfaced (REQUIREMENTS.md Out of Scope) |
</threat_model>
